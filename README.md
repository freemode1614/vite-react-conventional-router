## 支持的文件名与路由的映射

```
`src/pages/index.tsx`     -> `/`
`src/pages/home.tsx`      -> `/home`
`src/pages/home.@id.tsx`      -> `/home`
`src/pages/home.list.tsx` -> `/home/list`
`src/pages/home.list.$id.tsx` -> `/home/list`
```

## 如何组织页面文件内容

文件内容组织参考 react-router 的 [**lazy**](https://reactrouter.com/en/main/route/lazy#lazy)。
